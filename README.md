# band-name-generator-python

This is a beginner Python project created during my 100 Days of Code challenge.

The program asks the user:
- Which city they grew up in
- The name of a pet

It then combines both to generate a band name.

Example:
City: Boston
Pet: Milo

Band Name: Boston Milo

# What I learned 
- Basic user input with Python
- String concatenation
- How to structure a simple project
